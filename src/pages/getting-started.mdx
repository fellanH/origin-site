# Getting Started

Build your first Origin plugin — from zero to installed — in about five minutes.

---

## 1. Prerequisites

You'll need the following before you start:

- **Node.js 22+** — check with `node -v`
- **Origin desktop app** — [Download the latest release](https://github.com/fellanH/origin/releases), or install via Homebrew:

```bash
brew tap fellanH/origin
brew install --cask origin
```

<Callout type="tip">
  Already have an older Node version? We recommend using
  [fnm](https://github.com/Schniz/fnm) or [nvm](https://github.com/nvm-sh/nvm)
  to manage multiple versions.
</Callout>

---

## 2. Clone the starter

Grab the official plugin starter template:

```bash
gh repo clone fellanH/origin-plugin-starter my-plugin
cd my-plugin
```

Or with `git`:

```bash
git clone https://github.com/fellanH/origin-plugin-starter my-plugin
cd my-plugin
```

The starter includes a working `manifest.json`, a React component, and a Vite dev server pre-configured for Origin's plugin protocol.

---

## 3. Install and dev

Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

This starts a local server at `http://localhost:5173`. Origin can load your plugin directly from this URL during development.

To see it live, open Origin and go to **Settings → Developer → Add plugin URL**, then paste `http://localhost:5173`.

<Callout type="info">
  Hot module replacement works out of the box — save a file and see changes
  instantly in Origin.
</Callout>

---

## 4. Build

When you're ready to ship, create a production build:

```bash
npm run build
```

This compiles your plugin into the `dist/` folder — a self-contained bundle that Origin can load without a dev server.

<Callout type="warning">
  Make sure your `manifest.json` has a unique `id` field before building. Two
  plugins with the same ID will conflict.
</Callout>

---

## 5. Install into Origin

Install the built plugin into your local Origin app:

```bash
npm run install:plugin
```

This copies `dist/` into Origin's plugin directory. The plugin will be available immediately — no restart required.

---

## 6. See it work

1. Open **Origin**
2. Open the **Plugin Manager** (Settings → Plugins)
3. Find your plugin in the list and **enable** it
4. Press `Cmd+N` to open the launcher, then select your plugin

Your plugin is now running as a panel in Origin.
