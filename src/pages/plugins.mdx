# Plugin Gallery

Community-built plugins for Origin. Install any plugin from Settings → Plugins → Install.

## Official

### origin-plugin-starter

The canonical starter template. A minimal "Hello World" plugin that demonstrates the full plugin lifecycle.

- **Repo:** [github.com/fellanH/origin-plugin-starter](https://github.com/fellanH/origin-plugin-starter)
- **Author:** [@fellanH](https://github.com/fellanH)
- **License:** MIT

```bash
git clone https://github.com/fellanH/origin-plugin-starter
```

## Submit your plugin

Built something cool? Open a PR to add it to the gallery.

1. Fork [origin-site](https://github.com/fellanH/origin-site)
2. Add a card to `src/pages/plugins.mdx`
3. Submit a pull request

Cards should include: name, short description, repo link, author, and license.
